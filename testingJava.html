Objects

Objects are programmable things in javascript.
The syntax for changing a property of an object is object.property
Working with object methods syntax object.method()
Example:
history.back()

runs

const goBack = confirm("Do you want to go back?");
if (goBack === true){
history.back();
}

Events

An event is an action that occurs in response to some external stimulus, such as reloading, clicking, pressing a key, scrolling the page.
You need event listeners to listen for a specific event to happen and then it responds to that.
for specific listeners go to page 549

Arrays

To declare an array the syntax is const arrayName = new Array();
To declare and populate the array it is const arrayName = new Array(value1, value2, value3, ...);
You can run loops to populate the arrays aswell.
Arrays are very similar to python dictionaries
